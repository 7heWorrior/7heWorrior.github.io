Core Structures

ArrayList Random Access
List<T> l = new ArrayList<>();
l.add(v); l.get(i); // O(1) read
ArrayDeque Stack / Queue
Deque<T> d = new ArrayDeque<>();
d.push(v); d.pop(); // LIFO
d.offer(v); d.poll(); // FIFO
HashMap Key-Value
m.putIfAbsent(k, v);
m.getOrDefault(k, 0);
m.computeIfAbsent(k, x -> ...);
PriorityQueue Min/Max Heap
PQ<T> q = new PQ<>((a,b)->b-a);
q.offer(v); q.poll(); // O(log n)
